# Revit 소화배관 내진 버팀대 자동 배치 및 하중 검증 자동화 툴 개발

## 기간
2026년 2월(1개월)

## 목적
Revit 모델 내 소화 배관 데이터를 분석하여 시스템 및 유수 방향 기준으로 내진버팀대(종/횡/가지배관) 설치 지점을 산출하고, 각지점별 하중 지지력을 자동으로 계산하여 구조적 안전성을 검증하는 프로그램 개발

## 팀원
- 1명 (개발자)

## 사용 도구 및 기술
- **C#**
- **TortoiseSVN**
- **DevExpress**
- **Revit API**

## 주요 기능 및 구현 내용

- **배관망 분석 및 자동 분류**
  - 모델링된 배관의 System Type 및 물 흐름 방향(유수 방향)을 추출하여 전체 배관망을 구역별로 분류

- **기하학적 기준 확립**
  - 3D 공간상의 배관 각도, 기울기 여부를 판단하여 버팀대 배치를 위한 명확한 수학적/기하학적 기준 마련
  
- **버팀대 설치 지점(Point) 산출**
  - 구역과 구역의 배치 조건 및 거리 기준을 적용하여 종 버팀대 및 횡 버팀대가 설치될 정확한 Point 수집
  - 수집된 지점을 바탕으로 내진 버팀대 모델 자동 생성 및 배치
    
- **하중 계산 및 검증 로직 구현**
  - 각 버팀대마다 거리 기준에 따른 지지 가능 하중을 계산식으로 구현
  - 생성된 받침대가 요구되는 하중을 버틸 수 있는지 없는지 자동으로 판별하는 기능 구축

- **연산 속도 최적화 고민**
  - 수많은 배관 객체를 반복 탐색하고 조건을 비교하는 과정에서 모델링 생성 및 검증 시간을 단축하기 위한 데이터 처리 방식 개선
    
---

> 본 프로젝트는 3D 공간의 기하학적 데이터를 분석하고 대규모 객체의 반복 연산을 최적화하여, 복잡한 내진 버팀대 배치 및 검증 작업을 자동화하는 데 기여했습니다.
